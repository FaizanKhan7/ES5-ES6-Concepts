Difference between var const & let

var is function scoped and if we try to use a variable declared with var before the actual declaration, we'll get undefinied & if we try to reference a variable which is neither declared nor defined. When we try to reference such variable then the result will be not defined.

Const & let are blocked scoped and if we try to use a variable declared with let or const before the declaration, it will throw a reference error.

Hence, variables declared with let can be re-assigned, but variables declared with const cannot be reassigned.

Difference between function scope & block scope

In function scope, the variables and functions defined inside a function block are not accessible outside its block.

Variables declared with the var keyword cannot have Block Scope. A block scope is the area within if statements, switch conditions or for & while loops. Whenever we see {curly brackets}, it is a block. In ES6, const and let keywords allow us to declare variables in the block scope, which means those variables exist only within the corresponding block.

Difference between ES5 function & ES6 Arrow function

Basically, a JavaScript function is a block of code designed to perform a particular task. A JavaScript function is executed when "something" invokes it (calls it).

An arrow function expression is a alternative to a regular function expression.

The only difference between a regular function & arrow function is that, we don't use function keyword for arrow function & before we return the expression we add fat arrow =>.

Spread Operator & Rest Parameters

It was introduced in ES6, which makes it easy to merge arrays and objects. The spread operator takes an array (or any iterable) and spreads it values.

The rest parameter syntax allows us to represent an indefinite number of arguments as an array.

Destructuring

The destructuring assignment syntax is a JavaScript expression that makes it possible to unpack values from arrays, or properties from objects, into distinct variables.

higher order array methods

The higher order array methods are the powerful methods in JavaScript & these higher order array methods take in the function as a parameter and that callback function can take in multiple parameters. We can also chain on to other array methods. Their functional programming is very powerful & we can manipulate data however we want & its pretty crazy stuff.

Some of the important one's are:

  • forEach() - it just loops through an array.
  • map() - it allows us to create a new array from an array.
  • filter() - it will allow is to create a new array based on a condition.
  • reduce() - it executes a reducer function (that you provide) on each element of the array, resulting in single output value.
  • sort() - it sorts the elements of an array in place and returns the sorted array.
